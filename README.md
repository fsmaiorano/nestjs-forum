## About

This is a simple project that simulates a forum system. It is built with NodeJS.

## Architecture
Clean architecture - Work in progress

## Technologies Used
- NodeJS
- Typescript

## How to run

### Prerequisites
- NodeJS
- NPM
- Docker

### Steps
1. Clone the repository
2. Run `npm install`
3. Run `docker-compose up -d`
4. Run `npm run start:dev

## How to run tests

### Prisma
